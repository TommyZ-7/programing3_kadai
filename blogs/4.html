Copilotとは

GitHub CopilotはOpenAIが開発したAIアシスタントツールで入力したコードを自動で補完したり続きを生成したりできるものである。

様々な言語をサポートしており本サイトを作成する際もかなり役に立った。

このGitHub CopilotはVisual Studio Codeなどの一部の統合開発環境（IDE）で利用できる。

また、この機能を利用するにはGitHub Proに加入する必要がある。

GitHub Global Campus

GitHub Global Campusとは学生であれば誰でも参加できるプロジェクトであり、無料でGitHub Proの有料機能を使えるようになる。

また、GitHub Copilotも利用できるようになる。

申請に学生メールアドレスと学生証の手書きのコピーを作る必要があるが、学生ならぜひ参加してみてほしい。

注意点は手書きでないと加工画像とみなされ拒否られるので横着せずに手書きでやろう。
